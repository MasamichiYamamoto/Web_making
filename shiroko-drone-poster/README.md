# 白子ドローンスクール — B2ポスター案

三重県鈴鹿市・新校舎オープン告知のB2ポスター (515 × 728 mm) デザイン3案。

## ファイル構成

- `index.html` — メインのプレビューページ（3案を横並び表示）
- `posters.jsx` — 3案のReactコンポーネント
- `design-canvas.jsx` — Pan/Zoom対応のデザインキャンバスコンポーネント

## デザイン案

| Option | 方向性 | 特徴 |
|---|---|---|
| A | Editorial / Index | クリーム地・編集デザイン、巨大な「06」 |
| B | Bold Sky / Modernist | 空グラデーション＋幾何ドローン、紺×黄 |
| C | Tech Grid / Vertical | 黒地シアングリッド、縦組み漢字 |

## 表示方法

`index.html` をブラウザで開くだけ。外部CDNからReact / Babel / Google Fontsを読み込みます。

## サイズ

- B2: 515 × 728 mm (1px = 1mm 換算で 515 × 728 px)
- 印刷用にスケールアップする場合は、`posters.jsx` 冒頭の `W`, `H` を変更してください
